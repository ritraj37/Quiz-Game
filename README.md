# Quiz-Game
A simple Quiz Game built using HTML, CSS, and JavaScript where users answer multiple-choice questions and receive their score at the end. The game provides an interactive way to test knowledge with instant feedback.

# ❓ Quiz Game

The **Quiz Game** is a web-based application that allows users to test their knowledge by answering multiple-choice questions. The game displays questions one by one and calculates the final score based on correct answers.

This project demonstrates **JavaScript logic, DOM manipulation, and event handling**.

---

## 🚀 Features

* Multiple-choice quiz questions
* Instant feedback for answers
* Score calculation at the end of the quiz
* Simple and interactive user interface
* Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript** – Quiz logic and score calculation
